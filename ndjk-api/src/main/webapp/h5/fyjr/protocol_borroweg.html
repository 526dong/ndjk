第一条释义

  • 1、本协议：指本《立马有钱借款服务协议》、附件，以及借款人和出借人在服务方运营的平台账号、借款人在申请借款过程中同意的所有条款；

  • 3、协议生效时间:指出借人通过服务方的平台向借款人正式发放借款的时间；
  • 4、协议有效期间:指出借人向借款人发放借款，至借款人偿还所有的本金和利息并向服务方支付所有服务费的期间；
  • 5、期间:所有的期间均包括起止日期的当天；
  • 6、数字:所有数字均包括本数；
  • 7、个人信息：指可用于识别用户身份和信用的信息，包括但不限于身份证号、银行账户、电商账号、地理位置和电子通讯录等。

第三条各方权利与义务

  • 1、各方权利：
  • 1.1 出借人权利
    1. （1）出借人委托服务方于第三方支付平台上以出借人名义设立资金账户，并不可撤销地授权服务方使用资金账户上的资金进行审批、放款、借款回收及催收；
    2. （2）出借人有权向服务方了解借款人的信用评估进度和结果；
    3. （3）出借人有权查看借款人所提交并授权服务方向借款人披露的个人信息；
    4. （4）借款人出现第五条中的情形时，出借人授权服务方解除本协议，并授权服务方依据本协议向借款人主张权利；
    5. （5）出借人有权将本协议项下的全部或部分借款债权转让予第三方。
  • 1.2 借款人权利
    1. （1）借款人有权向服务方了解自己的信用评估进度和结果；
    2. （2）借款人不可撤销地授权服务方在第三方支付平台上以借款人名义设立资金账户，并不可撤销地授权服务方在出借人放款后，将约定服务费自资金账户内一次性扣收，且在借款人还款时将还款金额自银行账户内划转至出借人。
  • 1.3 服务方权利
    1. （1）服务方有权向借款人收取信审费、账户管理费等服务费；
    2. （2）服务方按照本协议的约定，受出借人的委托有权管理出借人资金、发放借款、回收及催收借款、划扣相关款项、解除本协议；受借款人的授权在借款人还款时从借款人的银行授权账户中划扣相应的款项, 无需另行经过借款人的同意或者提前通知借款人；
    3. （3）出借人与借款人同意并确认，服务方有权验证出借人和借款人提供的个人信息，包括但不限于身份证号、银行账户、电商账户、地理位置和电子通讯录等信息；
    4. （4）服务方有权根据对借款人信息和征信记录的审核情况决定是否代出借人发放借款；
    5. （5）借款人及出借人确认并同意，借款人不支付利息或不履行还款义务以及未完全履行前述义务时，服务方有权向出借人披露服务方所知晓的所有关于借款人的信息，同时服务方有权代出借人通过包括但不限于网络、报刊、电视、征信系统等方式向第三方披露借款人图像、姓名、手机号、身份证号码、逾期还款情况等信息。
  • 3.4 若甲方的还款金额不足时，清偿顺序为：若逾期两期以上，应首先清偿时间在前的借款债务，然后再清偿时间在后的借款债务；同一期借款债务的清偿顺序依次为：各种费用(不足的，按比例清偿)，违约金，逾期利息，利息，本金。

第四条：甲方的陈述与保证

  • 4.1 甲方保证并承诺借款真实、合法，并按照约定用途使用借贷资金，不会用于出借等其他目的，也不会将所借款项用于任何非法用途。
  • 4.2 甲方保证提供真实、准确、完整的个人信息及融资信息，其提供的其他资料也均为真实、完整、有效，并且甲方进一步保证其不会提供任何虚假信息或隐瞒重要事实。
  • 4.3 甲方承诺不会将本协议项下的债务以任何形式转让给任何其他方，如转让亦不对乙方及出借人发生任何法律上的效力。

第五条违约责任与救济

1、以下情况视为出借人违约：

  • （1）出借人在服务方填写的个人信息不真实的；
  • （2）出借人身份证号、通讯地址、联络方式等发生变更后，没有及时在服务方平台更新的；
  • （3）出借人所提供的借款来源不合法的；
  • 情形（1）（2）导致出借人的损失，由出借人自己承担责任；由（3）情形导致借款人受损失，由出借人承担借款人的直接损失。

2、以下情况视为借款人违约：

  • （1）借款人不能按照本协议约定履行还款义务的，包括不完全履行等情形；
  • （2）借款人提前偿还借款的；
    1. 出现情形（1）时，借款人应根据借款本金，按照2%/天的标准每日向出借人支付逾期违约金。
    2. 出现情形（2）时，借款人应当向出借人支付当期应还利息以及本金。
    3. 若出现以上情形（1）（2）时，借款人自愿授权服务方获取其最新的联系方式用以协助出借人进行还款提醒工作。
    4. 如果因借款人违约导致发生诉讼，应由借款人承担出借人和服务方的律师费用。

3、以下情况视为服务方违约：

  • （1）服务方平台系统故障，致使出借人应得的利息、借款本金不能追回的；
  • 由此情形导致出借人受损的，由服务方承担出借人的直接损失。

第六条协议的解除

1、以下情形，出借人授权服务方以电话告知的方式解除本协议，本协议自电话告知借款人后自动解除，协议解除后，服务方可代出借人向借款人主张权利。借款人授权服务方可以直接从借款人申请借款时绑定的借记卡中直接划扣相应借款数额，无需经过借款人同意。在代扣失败的情况下，借款人必须在本协议解除后五日内一次性支付以上款项：

  • （1）借款人未按本协议约定履行还款义务或向服务方支付管理费达三十日的；
  • （2）借款人在逾期还款后有逃避、拒绝沟通或者拒绝承认欠款事实等恶意行为的；
  • （3）借款人在合同期限内未到还款日前，根据出借人或者服务方的合理判断借款人已不具备归还借款能力；
  • （4）借款人提供虚假资料，或者隐瞒重要事实，或者身份证号、通讯地址、联络方式等发生变更后，未在变更后三日内及时通知服务方的；
  • （5）借款人向本协议外的第三人借款或为本协议外的第三人提供担保但是没有提前七日书面告知服务方的；。
  • （6）借款人的任何财产遭受没收、征用、查封、扣押、冻结等可能影响其履行能力的不利事件，且未及时对出借人和服务方进行书面通知或不能及时提供有效补救措施的；

第七条 刑事责任

若借款人在办理借款期间存在提供虚假资料或隐瞒重要事实，同时存在非法占有目的构成犯罪的，出借人有权向公安机关进行报案，同时出借人不可撤销的授权给富银金融信息服务（北京）有限公司代为报案。

第八条服务方的免责

以下情形，服务方免责：

  • 1、借款人拒不还款，且服务方披露了借款人的信息后难以追回借款的，服务方免责；
  • 2、其他依法不应由服务方承担责任的情形。
