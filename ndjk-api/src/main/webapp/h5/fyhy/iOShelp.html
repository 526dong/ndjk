产品相关问题

产品介绍

立马有钱是富银弘远旗下的一款简单便捷的无抵押纯信用借款产品，借款用户只需进行线上简单认证即可申请借款，认证一般3分钟即可完成。申请通过后，最快5分钟到账，操作简单，服务费用低。

注册或者更改密码时收不到验证码怎么办？

重启手机或清理手机缓存；退订过短信，请联系客服恢复短信发送。

如何更换手机号？

暂时还不支持用户自行更改注册手机号。

如何更换银行卡？

账户借款还清后，前往个人中心重新绑定银行卡。

打开手机白屏是怎么回事？

对于很多iOS用户，下载app之后打开app会发现没有数据，这是因为iOS10系统新增了无线助理功能，可以通过如下方式解决：设置-蜂窝移动网络-无线局域网助理-打开，再去登陆那些软件，当有弹窗要记得点允许就ok了。

借款相关问题

可借额度是多少？

立马有钱的借款额度为300元、400元、500元、600元、700元、800元、900元、1000元、1100元、1200元。

可借款期限是多少？

借款期限有7天、14天两种。

综合服务费？

综合服务费包括支付信审费、账户管理费。

目前还款方式有哪些？

我们会于还款日当天向您绑定的银行卡进行自动扣款，请您提前将资金准备充足。

逾期会收取费用吗？

平台会对逾期的费用收取逾期费。

不还款的影响有哪些？

不还款将会被记录个人征信，将会影响到您日后的购房贷款、购车贷款、甚至一切需要信用评分的服务申请。

认证相关问题

借款人需要满足什么条件？

年龄：18-45周岁(中国公民)；拥有稳定的收入来源；无不良的信用记录。

申请借款的流程？

用本人实名认证的手机号注册立马有钱->完成必要的认证->申请借款额度与期限->等待审核->等待打款->收到打款。

借款申请需要哪些认证？

身份证及人脸识别、个人信息认证、工作信息认证、紧急联系人认证、芝麻授权、手机运营商授权、银行卡认证。

为什么要提交这些资料？

为了防止别人盗用您的信息，并更精确的计算您的授信额度。我们不会将您的信息作为其他用途。会严格遵守监管部门规定，保护用户个人信息。

为什么会读取联系人失败？

由于您没有授权立马有钱读取联系人的权限。①iOS用户，可以通过设置->隐私->通讯录->立马有钱开启权限。

芝麻信用是必须的吗？

立马有钱产品目前要求用户必须有芝麻信用且授权立马有钱读取您的芝麻信用分数。否则无法完成借款。

服务密码错误是什么意思？

服务密码为手机运营商密码，错误或忘记可以询问服务商查询或更改。

手机运营商认证时提示“无法获取运营商信息”

该步骤将会认证您当前注册手机号码是否是本人在运营商（移动、联通、电信）实名的。即手机实名信息必须为本人，如需更换手机号，可以联系客服修改。也可能是因为您在运营商那里开启了账单保护功能，联系运营商关闭改功能即可。

审核相关问题

审核需要多久？

当天提交的审核保证当天处理，下午18点后提交的审核需要延顺到第二天工作时间处理。审核速度和当天的申请量相关，还请各位耐心等待。

审核通过后多久放款？

审核通过后2小时内打款，最快半小时内放款。

审核被拒绝的原因有哪些？

①输入信息有误，这种一般被视为假数据所以被拒绝可能性很大；② 认证不全面，比如没有芝麻授信之类，或者有些用户会先授信再取消授信，这种也会失败；③ 信用记录不好的用户一般也会被拒绝。

被拒绝后是否可以再次申请？

可以再次申请。

审核通过后的打款时间？

审核通过后一般30分钟内即可收到打款。
